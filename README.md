# Network-Traffic-Analysis-Lab
Overview

This report analyzes a malicious network traffic capture involving an Emotet and Trickbot co-infection using Wireshark, with additional threat intelligence from [VirusTotal](https://www.virustotal.com?utm_source=chatgpt.com) and [AbuseIPDB](https://www.abuseipdb.com?utm_source=chatgpt.com). The investigation identified multiple indicators of compromise (IOCs), including command-and-control (C2) communication, HTTP data exfiltration, SMB lateral movement, Active Directory enumeration, SSDP reconnaissance, and Domain Generation Algorithm (DGA) activity. Analysis of nearly 20,000 packets and numerous external connections revealed significant malicious behavior associated with advanced malware operations and potential ransomware deployment.
